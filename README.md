This script allows the user to switch between the Twitch and YouTube player without leaving the Twitch mod view page. It replaces the player with the chosen player ( Twitch or YouTube ) using a URL.

Entering a URL in the input box and clicking "Swap to YouTube" will switch the player to the YouTube player while allowing the user to stay on the Twitch channel mod view page. The button will switch from a YouTube icon to a Twitch icon indicating what mode the user is currently in.
